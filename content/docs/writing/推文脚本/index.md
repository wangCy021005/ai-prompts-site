# 推文脚本提示词

> **用途**：把小说精彩章节改成推文视频脚本，发抖音/快手拉新变现。
> **优先级**：可选放大器，**先写好小说再考虑这个**。周末花 2 小时做。

## 推文视频制作流程

```
小说精彩章节
   ↓ 提取 3-5 分钟解说稿
AI 配音（剪映/魔音工坊）
   ↓
配画面（解压视频/游戏录屏/动漫图）
   ↓ 剪映合成
发抖音/快手/B站
   ↓ 评论区引导
"搜《xxx》看全文" → 番茄拉新分成（6-12 元/个）
```

## 提示词 1：章节改解说稿

```
你是一位抖音小说推文博主，擅长把网文改成吸引人的解说稿。

【原文章节】
[粘贴小说精彩章节，约 4000 字]

请改成 3-5 分钟的推文解说稿，要求：
1. 开头 3 秒必须抓人（用最炸的冲突开场）
2. 用第三人称解说，"这个男人..."式口吻
3. 保留核心爽点，删掉细节描写
4. 每分钟留 1 个悬念，吊着观众
5. 结尾卡在最爽的地方，不要讲完
6. 引导语："想知道后续？搜索《书名》看全文"
7. 字数 800-1200 字（3-5 分钟语速）

输出纯解说稿。
```

## 提示词 2：开头钩子优化

推文开头 3 秒决定 80% 完播率，单独优化。

```
我有一段小说推文解说稿，开头不够抓人。

【原开头】
[粘贴前 100 字]

请重写 5 个版本的开头，每个版本用不同的钩子：
1. 冲突钩子：直接上最炸的矛盾
2. 反转钩子：先说结果再倒叙
3. 悬念钩子：抛出无法解释的现象
4. 共情钩子：让观众代入主角憋屈
5. 数据钩子：用夸张数字开场

每个版本 50 字以内，要让人忍不住看下去。
```

## 配画面素材建议

不用自己拍，用现成素材：

| 类型 | 适合题材 | 素材来源 |
|------|---------|---------|
| 解压视频（洗地毯/切肥皂/史莱姆）| 言情/年代文 | 抖音/B站搜"解压素材" |
| 游戏录屏（王者荣耀/我的世界）| 系统流/玄幻 | 自己录或素材站 |
| 动漫图轮播 | 所有题材 | 即梦/MJ 生成 |
| AI 生成画面 | 短剧化 | 即梦/可灵（成本高）|

**副业建议**：用解压视频或游戏录屏，零成本，流量还不错。

## 发布与拉新

### 发布平台
- 抖音（主）：流量最大
- 快手：下沉市场，年代文效果好
- B站：长解说，完播率高但拉新转化低
- 视频号：中老年用户，年代文适合

### 评论区引导话术
- 不要直接发链接（限流）
- 用"搜索《书名》"引导
- 置顶评论："书名《xxx》，番茄免费看全文"

### 拉新授权
- 只走官方：番茄推文、巨量星图
- 不要第三方"高价拉新"
- 一个新用户 6-12 元，爆款视频单条可带 500-2000 个

## 副业节奏建议

- **写小说优先**：日更不能停，推文是额外的
- 周末集中做：周六生成 3-5 条解说稿 + 合成视频，周日发布
- 矩阵号：做 3-5 个号，同一条视频不同号发（去重处理）
- 不要每天做：太累会拖垮写小说的主业

## 收益预期

| 情况 | 收益 |
|------|------|
| 单条爆款（百万播放） | 可带 500-2000 拉新，单条 3000-2 万 |
| 矩阵号（10 个号日更） | 稳定月 1-5 万 |
| 90% 的人 | 月入 <2000（没流量） |

⚠️ 推文是"劳动型收入"，停更就停钱。和小说（资产型收入）配合做。
