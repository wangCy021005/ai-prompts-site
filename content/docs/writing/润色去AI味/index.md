# 润色去 AI 味提示词

> **用途**：AI 生成正文后必须润色，删除 AI 味词汇，规避平台检测。
> **关键**：这是你作为"人"的核心价值，也是规避番茄 AI 检测的关键步骤。

## 高频 AI 味词汇黑名单

润色时重点搜这些词，能删就删，能换就换：

| 类型 | 词汇 |
|------|------|
| 套话 | 在这个世界中、不得不说、众所周知、殊不知 |
| 程度副词堆叠 | 不禁、不由得、不由自主、下意识地 |
| 视觉套话 | 映入眼帘、目光所及、映入视线 |
| 模糊词 | 仿佛、似乎、好像、宛如、犹如 |
| 转折滥用 | 然而、不过、但是、可是（每段都有就有问题） |
| 总结句 | 这一刻他明白了、他知道事情没那么简单 |
| 形容词堆叠 | 巨大的、恐怖的、惊人的、不可思议的 |

## 润色提示词

```
请润色以下网文正文，去除 AI 味，要求：

1. 删除/替换 AI 味词汇（不禁、映入眼帘、仿佛、然而等）
2. 长句拆短，超过 30 字的句子尽量拆
3. 对话改口语化，去掉书面语感
4. 删除无意义的心理独白和环境描写
5. 保持原意和爽点不变
6. 不要加新的剧情

【原文】
[粘贴 AI 生成的正文]

直接输出润色后的正文。
```

## 人工润色清单（AI 润完再自查）

- [ ] 有没有连续 3 段以上没有对话？（加对话）
- [ ] 有没有超过 50 字的长句？（拆短）
- [ ] 反派是不是太脸谱化？（加 1 句反派的算计）
- [ ] 爽点够不够明显？（不够就强化主角的得意/对方的震惊）
- [ ] 结尾钩子够不够？（不够就加个悬念）
- [ ] 角色名有没有写错/混用？

## 进阶：注入个人风格

润色时可以有意识地加入"签名手法"，让 AI 检测更难识别：

- 固定一个口头禅给某个配角
- 主角的内心吐槽用固定句式
- 某类场景用固定的转场方式
- 这些"小习惯"是 AI 难以复制的个人痕迹

## 时间分配建议

副业每天 30-45 分钟，建议：
- AI 生成正文：10 分钟
- **润色：20 分钟（重点投入这里）**
- 发布：5 分钟

润色占大头，因为这是你区别于"纯 AI 写手"的核心壁垒。

## 为什么必须润色

⚠️ **番茄/抖音都在升级 AI 检测**，纯 AI 不润色直接发会被限流甚至封号。

⚠️ **版权证据**：你润色后的版本是"人工参与创作"的证据，git 提交记录可追溯。

⚠️ **读者体验**：AI 味重的文读者读着累，留存率低。
