# ✍️ 写作类提示词

网文写作、内容创作相关提示词。来自实际写小说的实操经验，不是空泛模板。

## 提示词列表

- [小说续写提示词](./小说续写/) — 日更网文续写，保持爽点节奏
- [大纲生成提示词](./大纲生成/) — 主角设定、30 章大纲、单章细纲
- [润色去 AI 味提示词](./润色去AI味/) — 删除 AI 味词汇，规避平台检测
- [推文脚本提示词](./推文脚本/) — 小说改推文视频脚本，拉新变现

## 完整写作流程

```
1. 大纲生成    →  定主角、金手指、30 章大纲
2. 小说续写    →  每天日更正文
3. 润色去AI味  →  每章必做，规避检测
4. 推文脚本    →  数据起来后，周末做推文视频
```

## 使用建议

1. 提示词里的方括号 `[xxx]` 是需要你替换的变量
2. 不要照搬，根据自己的题材调整
3. AI 生成后必须人工润色（这是关键）
4. 日更比什么都重要，断更 3 天流量就掉
